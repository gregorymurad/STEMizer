Since Python is an interpreted language, you do not need to compile it. The interpreter will go from top to bottom and interprets each line of code. Let's define other variables, assign values and print these values:

A list in Python is a mutable and ordered data structure. What does this mean? It means that you can modify, delete and values to any position and each element has an index that is defined by a definite sequence of natural numbers. The values of a list are inside the square brackets []. Lists can contain duplicate values. 

teachers = ['greg', 'leonardo', 'giri']

print(type(students))

A for loop is used to iterate over a sequence (it can be a list, a tuple, a dictionary, a set or even a string) and can be written as follows: 

for i in teachers:
    print("Hello, " + i.title() + "!")

Classes in Python are pretty straightforward: 

import datetime
from abc import abstractmethod, ABCMeta

class FIU_Person:
    __metaclass__ = ABCMeta

    colors = "Blue and Gold"
    def __init__(self, name,pantherID,first_year,age):
        self.name = name
        self.pantherID = pantherID
        self.first_year = first_year
        self.age = age

    @abstractmethod
    def whatDoIDo(self):
        pass

    def whoIsMe(self):
        print("My name is " + self.name)

    def howLongAtFIU(self):
        x = datetime.datetime.now().year
        if x-self.first_year > 1:
            print("I have been at FIU for", str(x-self.first_year),"years.")
        else:
            print("I have been at FIU for", str(x-self.first_year),"year.")

class Student(FIU_Person):
    def __init__(self,degree,name,pantherID,first_year,age):
        self.degree = degree
        super().__init__(name,pantherID,first_year,age)

    def whatDoIDo(self):
        print("I study {0}.".format(self.degree))

class Professor(FIU_Person):
    def __init__(self, college, name, pantherID, first_year, age):
        self.college = college
        super().__init__(name, pantherID, first_year, age)

    def whatDoIDo(self):
        print("I work at {0}.".format(self.college))
